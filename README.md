# 网盘前端代码

## 项目设置
```
npm install
```

### 编译并热重载以进行开发
```
npm run serve
```

### 为生产环境编译和压缩
```
npm run build
```

### 检查并修复文件
```
npm run lint
```

### 项目启动
````
#先运行npm install

npm install

#然后运行npm run serve然后根据配置文件打开相应网页，比如我配置为8081端口则打开http://localhost:8080/

npm run serve
````
